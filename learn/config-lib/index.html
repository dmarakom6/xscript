添加标准库源文件

编写文件

xscript并不能很好使用python的源文件, 要适当重写.

如何重写:

  • 重写函数调用
  • xscript不能创建和调用python对象及其方法和变量, 请将创建对象的代码转换为一个返回该对象的函数, 添加函数以便使用对象的方法
  • 注意对传入参数的转换

贴示

在使用python库的时候应该对使用的库名进行更改: 比如os换成_os

使用

请在xscriptlib/__init__.py中添加一行来import刚刚编写的库.

以一个foo为例子:

  1. 编辑xscriptlib/__init__.py
  2. 输入import lib.foo as foo
  3. 重启解释器
  4. 编写程序测试一下
  5. 完成
