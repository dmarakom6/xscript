使用debug

debug命令是内嵌在解释器核心中的简单调试器

启动debug

xscript有一独立命令debug来启动debug. 会以横跨终端两段的=字符以及在=中居中的start debug和end debug来区分程序运行输出和debug输出.

贴示

在不支持os.get_terminal_size的平台上, 会以xscript: start debug和xscript: end debug来分隔

debug命令

  • exit - 退出debug

  • now - 显示现在位于的块

    now会读取core.XScriptInterpreter.block中的数据, 以.分隔

  • ls - 显示所有变量

  • show $name - 只显示名字为name的变量
